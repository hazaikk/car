API 文档

概述

本API提供汽车数据的访问接口，支持查询车型信息、价格、销量等数据。所有请求都需要进行身份验证。

认证

API使用Token认证，请在请求头中添加：

Authorization: Token your-token-here

API 端点

车型信息

端点 方法 描述
/api/v1/cars/ GET 获取车型列表
/api/v1/cars/{id}/ GET 获取特定车型详情

价格数据

端点 方法 描述
/api/v1/prices/ GET 获取价格数据
/api/v1/prices/trends/ GET 获取价格趋势数据

销量数据

端点 方法 描述
/api/v1/sales/ GET 获取销量数据
/api/v1/sales/analysis/ GET 获取销量分析数据
